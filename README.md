# Meu Portfolio

## Sobre
Portfolio pessoal hospedado em **felipecavalari.com.br** na hostinger

## Stack
- HTML + CSS + JavaScript
- Tailwind CSS (CDN)
- GSAP para animações
- Font Awesome para ícones



## Notas
- Hero usa layout split com typewriter multi-texto
- Skills usa marquee infinito (2 fileiras)
- About tem code window estilo terminal
- Mobile menu é overlay fullscreen


## Deploy
Deploy automático feito na hostinger ao realizar git push
